# Commercial Value Summary

## Target customers

- Mobile money operators
- Banks with mobile wallets
- Payment service providers
- Fraud operations teams and agent networks

## Core value

- Stop high-risk payouts before funds move
- Give agents better guidance at the exact decision point
- Reduce manual review volume by filtering clean transactions faster
- Improve customer trust after takeover attempts

## Business case

- Fraud prevention has direct financial impact.
- Operator-side trust tooling is easier to justify when it reduces losses, chargebacks, and support burden.
- CAMARA-based trust signals create a more defensible product than rules based only on app-side data.

## Go-to-market angle

- Start with mobile money fraud prevention for agent-assisted transactions.
- Expand into onboarding, passwordless recovery, and telecom-backed risk scoring for fintech platforms.
